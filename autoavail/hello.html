    Settings

    Gmail Integration

    Type {availability} in any Gmail compose window and it will be replaced with your free times for the next week.

    AI Email Generation

    When you use the {availability} tag, AutoAvail can use OpenAI to generate a professional email response with your availability.

    Enter your OpenAI API key to enable AI email generation:

    How to get an OpenAI API key:

    1. Visit OpenAI Platform
    2. Create an account or sign in
    3. Go to API Keys section
    4. Create a new API key (Make sure you have credits $$)
    5. Copy and paste it above

    Your API key should start with "sk-..."

    After saving your API key, reload Gmail tabs for changes to take effect.

    Troubleshooting

    If the Gmail integration isn't working:

    • Make sure you've authenticated with Google Calendar
    • Try refreshing your Gmail tab
    • Open the browser console (F12) to see debug messages
    • Try typing {availability} followed by a space
    • Check that your OpenAI API key is entered correctly

    For technical issues, check the browser console (F12 > Console tab) for detailed logs.

    If you see OpenAI API errors:

    • Ensure you're using a valid API key from OpenAI Platform
    • Make sure your API key is active and has not expired
    • The extension will still work with formatted availability data even if AI generation fails
